# #12906 같은 숫자는 싫어 (Lv1)

🔗 원문 링크  
https://school.programmers.co.kr/learn/courses/30/lessons/12906

---

## 📝 문제 설명

배열 arr가 주어집니다.  
배열 arr의 각 원소는 숫자 0부터 9까지로 이루어져 있습니다.

이때, 연속적으로 나타나는 숫자는 하나만 남기고  
전부 제거하려고 합니다.

단, 제거된 후 남은 수들을 반환할 때는  
배열 arr의 원소들의 순서를 유지해야 합니다.

---

## 📌 제한사항
- 배열 arr의 크기 : 1 이상 1,000,000 이하
- 배열 arr의 원소 : 0 이상 9 이하의 정수

---

## 📌 입출력 예

| arr | return |
|------|--------|
| [1,1,3,3,0,1,1] | [1,3,0,1] |
| [4,4,4,3,3] | [4,3] |
