## 📚 Hash (Dict Only) 문제 정리
- Counter로 대부분 Hash 문제를 해결할 수 있지만,
아래 문제들은 "개수"가 아닌 
"순서 / 관계 / 조합"을 다루기 때문에 Counter로는 해결 불가!
- 따라서 Dict(해시)를 활용한 풀이가 필요
- Counter : "개수 세기" 전용
- Dict : 위치 / 관계 / 조합 / 상태 변화 처리
